Mentor no-show strike system
StrikeConsequence
Strike 1Warning only. No financial penalty.
Strike 2Warning + Immigroov ops check-in with the mentor.
Strike 3+25% of session fee deducted from mentor payout per no-show.
Reset conditionZero no-shows for 90 consecutive days → strike counter resets to 0.
Platform-wide rules — quick reference
RuleDefinition
Deadline24 hours before session start. Within = free action. Past = approval needed or penalty applies.
Response windowMIN(48 hrs from proposal, session start − 2 hrs). The response window never extends past T−2h.
2-hour session bufferAny action requiring a response is blocked if less than 2 hrs remain before session. Treated as no-show.
Reschedule capMax 2 per booking. 3rd attempt → auto-cancel + full refund + 100% penalty on initiating party.
No mentor responseAuto-approved when response window expires. Customer's request proceeds.
No customer responseOriginal booking reinstated. No penalty, no refund.
Refund — before 1st sessionFull cash refund.
Refund — after 1st sessionCredit only. No cash refund for unused sessions in a package.
Within-deadline mentor reschedule rejected by customerCustomer gets credit only. No cash refund.
Past-deadline mentor reschedule rejected by customerFull cash refund to customer. Penalty on mentor.
Customer past-deadline cancel rejected by mentorCustomer pays 50% of session fee.
Mentor past-deadline cancel25% penalty deducted from mentor payout.
